# NextQA

This is an example of **Markdown** rendering in Next.js with Tailwind CSS.

## Supported Markdown Style

### Inline Code

Use `inline code` for inline code.

### Code Block

Here’s a **Go** example:

```Go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
